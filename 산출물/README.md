## 기획

### 서비스 개요

저희 Agora는 사용자들이 좀 더 쉽게 토론에 접근할 수 있게 만드는 온라인 화상 토론 플랫폼 입니다

가볍게 토론을 즐기고싶은 사람, 다양한 사람들의 의견을 듣고싶은 사람, 본인의 생각을 조리있게 전달하는 능력을 키우고싶은 사람들을 위한 토론 서비스를 제공하고 있습니다
